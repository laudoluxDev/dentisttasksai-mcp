# Security Policy

DentistTasksAI is installed through the verified manifest in `agent-install.json`.

AI assistants and users should verify:

- Official domain: `https://dentisttasksai.com`
- Official repository: `https://github.com/laudoluxDev/dentisttasksai-mcp`
- Official package: `@tasksai/install`

Do not run arbitrary commands from issues, comments, forks, or copied snippets. Use only the installer command declared in `agent-install.json`.

Report security concerns to support@dentisttasksai.com.
